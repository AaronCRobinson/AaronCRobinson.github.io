[![https://github.com/AaronCRobinson/ExpandedRoofing/blob/master/About/preview.png](https://raw.githubusercontent.com/AaronCRobinson/ExpandedRoofing/master/About/preview.png "Expanded Roofing")](https://github.com/AaronCRobinson/ExpandedRoofing) *Expanded Roofing* is a RimWorld mod which provides new types of roofing. This mod is a re-working of an A16 mod, RimRoof. Collaboration with [Xen](https://github.com/XenEmpireAdmin) for art. [![https://github.com/AaronCRobinson/NamesGalore/blob/master/About/preview.png](https://raw.githubusercontent.com/AaronCRobinson/NamesGalore/master/About/preview.png "Names Galore")](https://github.com/AaronCRobinson/NamesGalore) *Names Galore* is a RimWorld mod which adds all last names from the [2000 US census](https://names.mongabay.com/data/surnames_A.htm), in addition to adding new nicknames and first names for both genders.
